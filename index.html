Mi proceso de trabajo

1.- Análisis de usuario y benchmark

Un buen proyecto comienza haciendo un análisis de usuario para definir el público objetivo y la identidad de la marca (Solo si esto no está definido antes). Luego se hace un análisis comparativo de distintos referentes, para saber qué hacen bien y qué cosas están demás.

2.- Brainstorming y propuestas de diseño

Una vez realizados los análisis, hago una lluvia de ideas para poder realizar múltiples propuestas de diseño y otorgar múltiples soluciones. Teniendo las ideas solidificadas, realizo propuestas de diseño al cliente para llegar a un acuerdo para decidir cuál es la que le parece mejor.

3.- Prototipado del diseño

Para entregar una mejor idea de cómo será el sitio al final, se realiza un prototipo del diseño propuesto, con funcionalidades y efectos visuales en este. Se realizan testeos de funcionalidad con múltiples usuarios para luego corregir posibles errores o fallas en el diseño.

4.- Programación del sitio

Luego de ser testeado el prototipo, se comienza con la programación del sitio, replicando directamente la maqueta utilizada.

5.- Testeo y correcciones finales

Al terminar de programar, se sube el sitio web al hosting correspondiente y se asocia al dominio solicitado. Luego se prueba el sitio web estando en linea para posteriormente realizar las correcciones finales.

6.- Entrega

Cuando se comprueba que el funcionamiento del sitio es el correcto, se entrega el sitio web al cliente, junto con el hosting y dominio.
